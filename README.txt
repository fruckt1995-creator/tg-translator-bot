# Telegram Translator Bot (v2)

## Що вміє
- Автокорекція (LanguageTool) + виправлення мішаних літер
- Переклад у мову користувача Telegram або у налаштовану через /setlang
- Працює в приваті та групах (disable privacy у @BotFather)
- Команди: /setlang <код>, /mylang, /help

## Запуск
1) Встанови залежності:
   npm install
2) Додай токен у .env:
   BOT_TOKEN=...тут_токен...
3) Запусти:
   npm start

## Групи
- У @BotFather: /setprivacy → Disable
- Додай бота до групи
